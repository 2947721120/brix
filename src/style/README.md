## Brix Style -- 基础样式
+ brix.less 是主入口文件
    - base.less        继承KISSY Reset，减少推广阻力。同时做些更新
    - variables.less   全局的颜色配置，栅格变量
        - gray colors
        - accent colors
        - iconfont
        - zindex
        - grid
    - mixins.less      提供给开发者的mixins函数
        - radius
    - grid.less        栅格
    - type.less        字体
    - icons.less       iconfont
    - buttons.less     按钮
    - utilities.less   小工具
